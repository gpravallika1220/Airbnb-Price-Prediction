# Dataset Sources

This project uses the public **US Airbnb Open Data** from Kaggle.

## 1. Airbnb Listings — United States 2020
- **Filename:** `AB_US_2020.csv`  
- **Source (Kaggle):**  
  https://www.kaggle.com/datasets/kritikseth/us-airbnb-open-data?select=AB_US_2020.csv

## 2. Airbnb Listings — United States 2023
- **Filename:** `AB_US_2023.csv`  
- **Source (Kaggle):**  
  https://www.kaggle.com/datasets/kritikseth/us-airbnb-open-data?select=AB_US_2023.csv
